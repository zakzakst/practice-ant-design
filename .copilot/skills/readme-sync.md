### 【目的】
README.md を、現在のコード状態と乖離しないように整備する。

### 【参照してよい情報】
- README.md
- package.json
- 設定ファイル（存在する場合）
- ディレクトリ構成

### 【作業内容】
1. README.md の既存構成を確認する
2. 明示的に確認できる情報のみを抽出する
3. README に次のセクションが存在しない場合のみ追記する
   - 使用技術
   - 起動方法
4. 既存文章は意味を変えず、表現の調整のみ許可する

### 【禁止事項】
- 推測による情報追加
- セクションの削除
- 内容が不明な項目の追記

### 【出力形式】
- 変更内容を diff 形式で提示する
- 変更理由を簡潔に説明する

### 【重要】
コードの変更は行わないこと。
